# WMover

macOS 메뉴바 상주 윈도우 관리 유틸리티. 키보드 조합 + 마우스 드래그로 어떤 창이든 옮기고
크기를 조절하며, 카페인 스타일 **잠자기 방지** 기능을 함께 제공한다.

## 기능

- **창 이동**: `⌃⇧`(Control + Shift) + 좌클릭 드래그
- **창 크기조절**: `⌃⌘`(Control + Command) + 좌클릭 드래그 (최소 120×80)
- **잠자기 방지**: 앱이 켜져 있는 동안 맥이 잠들지 않게 한다 (화면 + 시스템).
  실행 시 자동으로 켜지며, 메뉴바에서 끄고 켤 수 있다.

## 빌드 & 실행

```bash
./build.sh
open WMover.app
```

- macOS 13.0 이상
- 첫 실행 시 **손쉬운 사용(접근성)** 권한 허용 필요 (창 이동/크기조절 기능에 필요).
  메뉴바 → "접근성 설정 열기…"로 바로 이동 가능. 잠자기 방지는 권한 없이 동작.
- 리빌드 시 바이너리가 바뀌어 접근성 권한이 초기화되므로 다시 허용해야 한다.

## 구조

- `Sources/WMover/main.swift` — 전체 구현 (단일 파일, AppKit + IOKit)
- `build.sh` — `swiftc`로 `WMover.app` 번들 생성 + ad-hoc 코드 서명
